[# Ball-Fight](https://jyo1205.github.io/Ball-Fight/)



볼 파이트(Ball Fight)
장르: 로컬 PvP 자동전투 전략


시놉시스
두 명의 플레이어가 제한된 예산 내에서 무기와 특성을 선택하여 자신만의 전투 빌드를 구성한 뒤, 자동으로 진행되는 전투를 통해 승패를 겨루는 2인 대전 게임. 플레이어는 다양한 무기와 특성 조합을 통해 전략적인 선택을 할 수 있으며, 각 조합의 성능을 비교하고 최적의 빌드를 탐색할 수 있다.


게임 특징
1. 예산 기반 빌드 시스템

플레이어는 정해진 예산 안에서 무기와 특성을 선택해야 하므로 전략적인 선택이 요구된다.

2. 다양한 무기 선택

Sword, Spear, Hammer 등 서로 다른 성능을 가진 무기를 선택할 수 있다.

3. 특성 강화 시스템

체력, 이동속도, 공격력 등의 특성을 선택하여 캐릭터를 강화할 수 있다.

4. 자동 전투 방식

전투가 시작되면 캐릭터가 자동으로 움직이며 전투를 진행한다.

5. 빌드 비교 및 재대전

다양한 조합을 시도하고 리매치 기능을 통해 여러 빌드를 반복적으로 비교할 수 있다.
